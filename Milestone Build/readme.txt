This is where the exported Milestone build should be placed once the code freeze and testing has been completed before each milestone.

Before Each milestone 
1. Right click on all files in this directory and select to delete them through the git interface
2. Commit those changes
3. Copy a tested, working, exported build of the app/game to this directory
4. Right click on the folder and select to add files using the Git interface
5. Commit the build and push it to the remote repo with a comment defining the milestone it is intended for
