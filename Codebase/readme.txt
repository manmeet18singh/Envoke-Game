The codebase folder is where you put the project files that you are working on. 
For the sake of an app development that would contain things like the android studio project.
For game development this is where the unity project should be located.

Commit and push the project files to the remote repo

1. Right click on the folder and select to add files using the Git interface
2. Commit the build and push it to the remote repo with a comment defining it as the first project commit
